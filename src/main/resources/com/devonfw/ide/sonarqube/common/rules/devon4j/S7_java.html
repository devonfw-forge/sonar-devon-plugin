This rule checks that base scope does not depend on base scope of another component part. Allowed dependencies from base scope:

  • Api scope of the same component
  • Api scope of a different component
  • Base scope of a different component

Noncompliant Code Example

    package com.devonfw.ide.sonarqube.component1.logic.base;

    import com.devonfw.ide.sonarqube.component2.logic.base.Bar;

    public class MyClass {}
